# Тестовое задание на позицию Middle Python backend developer
## Сервис скачивания и анализа текстовых файлов 

Асинхронный веб-сервис на FastAPI для автоматизированного скачивания архивов с текстовыми файлами через сторонний API, их распаковки, анализа содержимого (подсчет количества цифр) и сохранения результатов в базу данных.

---

## Стек

* **Backend:** Python 3.13+, FastAPI, Uvicorn
* **Database:** SQLite + aiosqlite 
* **HTTP Client:** `httpx` (с обработкой `Rate Limits` / HTTP 429)
* **Frontend:** Jinja2, HTML, JavaScript (Fetch API / Polling)
* **DevOps & Tools:** Docker, Docker Compose, Ruff

---

## Ключевые возможности

1. **Фоновый процессинг:** Скачивание файлов происходит в фоновом режиме через `BackgroundTasks`, не блокируя интерфейс.
2. **Устойчивость к Rate Limits:** При получении ответов `429 Too Many Requests` или `403 Forbidden` клиент считывает заголовок `Retry-After`, выдерживает паузу и повторяет запрос.
3. **Пакетная обработка:** Скачивание файлов происходит батчами (по 3 файла в ZIP-архиве) с последующим подтверждением получения на стороннем сервере.
4. **Аналитика:** В каждом текстовом файле ведется подсчет каждой цифры от `0` до `9`.
5. **Удобный UI:**
   * **Главная страница (`/`):** Индикация статуса скачивания в реальном времени, подсчет найденных/скачанных файлов и фиксация времени старта по новосибирскому часовому поясу (`Asia/Novosibirsk`).
   * **Страница реестра (`/files`):** Список скачанных файлов с сортировкой по убыванию времени, пагинацией и калькулятором суммы цифр по выборке (выбранные файлы, текущая страница или вся база).

## Запуск через Docker
### 1. Создайте и перейдите в директорию с проектом ###
```bash
mkdir project
cd project
```

### 2. Клонируйте проект с github ###
```bash
git clone https://github.com/Gleb4ik2001/installBiz-test-task
```

### 3. Создайте файл .env по примеру .env.example ###
```ini
TARGET_API_URL=http://91.199.149.128:18001
CANDIDATE_ID=Jj5Jf54JHa4558A
DOWNLOAD_DIR=downloaded_files
DB_PATH=data_db/database.db
```

### 4. Запустите Docker контейнеры ###
```bash
docker compose up -d --build
```

### 5. Отройте [http://127.0.0.1:8000/](http://127.0.0.1:8000/) в браузере ###

### 6. Остановите Docker контейнеры ###
```bash
docker compose down
```
